<details>

<br/>

- Intégration d'APIs métier (Pennylane, Toggl, Géorisques, Pappers…)

- Synchronisation multi-outils (CRM ↔ facturation ↔ gestion de projet)

</details>

<details>
<summary><b>🗄️ Architecture de données Airtable</b></summary>

</details>
